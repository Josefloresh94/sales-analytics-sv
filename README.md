# 🇸🇻 Sales Analytics SV

Dashboard de análisis de ventas para el mercado salvadoreño.

## Stack
- **Análisis:** Python, Pandas, Jupyter Notebook
- **Visualización:** Power BI, Angular + Chart.js
- **Backend:** FastAPI + PostgreSQL
- **Frontend:** Angular 17

## Estructura del proyecto
- `backend/data` — datasets generados
- `backend/analysis` — notebooks de análisis
- `backend/api` — API REST con FastAPI
- `frontend` — dashboard web en Angular
